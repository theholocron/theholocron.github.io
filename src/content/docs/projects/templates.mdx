A set of templates to help jump off from when building out new tools.

-   🧱 [base-template](https://github.com/theholocron/base-template/) – Lean project starter with batteries included—perfect for any stack.
-   🔧 [node-template](https://github.com/theholocron/node-template/) – Production-ready boilerplate with best practices and automation.
-   🖥 [cli-template](https://github.com/theholocron/cli-template/) – Modern scaffold for building polished Node.js CLIs with zero friction.
-   ⚛️ [react-template](https://github.com/theholocron/react-template/) – Clean, scalable boilerplate with modern tooling and dev ergonomics.
-   🔥 [nextjs-template](https://github.com/theholocron/nextjs-template/) – Full-featured Next.js starter with TypeScript, testing, and CI/CD built in.
-   🧩 [monorepo-template](https://github.com/theholocron/monorepo-template/) – Scalable with TypeScript, workspaces, and automated releases.

## 📦 Installation

The following pattern will generate a new repo based upon the provided template; swap out `<template>` for anyone of the templates listed below.

```
https://github.com/new?template_name=<template>&template_owner=theholocron
```

📝 Note: After cloning, make sure to run the [bootstrap command](../holocron/bootstrap/) to replace the template name and set up your project properly.
