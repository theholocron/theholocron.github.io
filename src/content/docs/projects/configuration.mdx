A monorepo containing all of our shareable configuration files.

-   [browserslist-config](https://github.com/theholocron/configs/blob/main/packages/browserslist-config/README.md) - A [Browserslist](https://browsersl.ist) configuration for all browsers and devices supported.
-   [bundlewatch-config](https://github.com/theholocron/configs/blob/main/packages/bundlewatch-config/README.md) - A [Bundlewatch](https://bundlewatch.io/#/) configuration for for monitoring package scope creep.
-   [commitlint-config](https://github.com/theholocron/configs/blob/main/packages/commitlint-config/README.md) - A [CommitLint](https://commitlint.js.org) configuration for writing Git commits.
-   [eslint-config](https://github.com/theholocron/configs/blob/main/packages/eslint-config/README.md) - [An ESLint](https://eslint.org) configuration for writing well-formed Javascript.
-   [jest-config](https://github.com/theholocron/configs/blob/main/packages/jest-config/README.md) - A [Jest](https://jestjs.io) preset for testing code.
-   [lint-staged-config](https://github.com/theholocron/configs/blob/main/packages/lint-staged-config/README.md) - A [Lint Staged](https://github.com/lint-staged/lint-staged) configuration for linting code that has been staged in Git.
-   [prettier-config](https://github.com/theholocron/configs/blob/main/packages/prettier-config/README.md) - A [Prettier](https://prettier.io) configuration for formatting libraries.
-   [tsconfig](https://github.com/theholocron/configs/blob/main/packages/tsconfig/README.md) - A [TypeScript](https://www.typescriptlang.org) configuration for checking types.
